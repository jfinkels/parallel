## Getting and compiling this work ##

To get a copy of the most recent version of this work:

    git clone git@github.com:jfinkels/parallel.git

To compile this work:

    cd parallel
    pdflatex parallel
    bibtex parallel
    pdflatex parallel
    pdflatex parallel
